    Tic Tac Toe in Python

    Written in Python and using the basic Turtle library for the GUI. While writing this as a one player game, I got to explore the basic logic for an AI opponent.


    Tip Calculator App in C# (made with Xamarin)

    Type in your bill amount, use the slider to select your tip percentage and how many people joined you for dinner and the app will calculate the tip amount!
    Using a web API you can also convert the bill into three select currencies.


    CoViduliser App in C# (made with Xamarin)

    Using the free web API 'About Corona', get up to date info on the world and your chosen country's COVID-19 stats.
